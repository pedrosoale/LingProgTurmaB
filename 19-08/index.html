É possível enfatizar uma palavra com o elemento em, dar mais ainda mais ênfase com o elemento strong, ou marcar uma palavra com o elemento mark.

Também podemos demarcar código fonte com o elemento code.

Podemos forçar uma quebra de linha aqui
com o elemento br.

Podemos usar sub para subscrito, como em H2O, ou sup para sobrescrito em y = x2 + 3x.

Imagens são inseridas com o elemento img.

O elemento a corresponde a um link capivara.

Também podemos abrir link em nova aba com o atributo target = "_blank". Capivara.
